angular-color-this {{selectedText}}

A super light AngularJS module for adding colors to anything!

Installation

Install with bower bower install angular-color-this --save

Source in your HTML <script src="bower_components/angular-color-this/dist/angular-color-this.min.js"></script>

Inject into your angular app angualr.module('myApp', ['ngColorThis'])

Usage

Labels

<li ng-repeat="day in days">
  <span class="label label-info" color-this="background-color" data-color="day">{{day}}</span>
</li>
  • {{day}}
  • {{month}}

Borders

<img src="..." alt="..." color-this="border-left-color" data-color="selectedText">
